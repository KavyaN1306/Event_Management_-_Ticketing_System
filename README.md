# Event Management & Ticketing System (MongoDB)

## Description
This project implements a MongoDB-based backend design for an Event Management and Ticketing System.  
It focuses on schema design, sample data, CRUD operations, aggregation pipelines, and indexing.

## Collections Used
- Users
- Events
- Tickets
- Categories

## Features Implemented
- Creation of MongoDB collections
- Insertion of sample documents
- CRUD operations on events
- Aggregation pipelines for reports such as:
  - Top events by ticket sales
  - Revenue per event
  - Number of attendees per event
- Indexing for performance and data integrity
